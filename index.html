Projets

Voici un aperçu des projets auxquels j’ai participé au cours de ma formation. Ces projets m’ont permis de mettre en pratique mes connaissances en programmation, de travailler en équipe et de développer des solutions concrètes.

Simulateur de guichet automatique

Application native simulant le fonctionnement d’un guichet automatique, incluant la gestion des clients, des comptes et des transactions bancaires.

Technologies : Java, JavaFX

Gestion d’activités de groupes

Application web permettant à des groupes d’amis de créer, gérer et organiser des activités avec une gestion complète des membres et des rôles.

Technologies : PHP, JavaScript, HTML, CSS

Green Pulse

Green Pulse est une entreprise canadienne qui développe une plateforme web automatisée afin de connecter directement les producteurs de légumineuses aux acheteurs internationaux, réduisant les intermédiaires et optimisant les transactions.

Technologies : Thymeleaf, Bootstrap, JavaScript, JQuery, Ajax, SpringBoot

Kaypic Communication – Plateforme de messagerie et de collaboration

Application web de communication en temps réel inspirée de Microsoft Teams, destinée à connecter des coachs, joueurs, parents et administrateurs. L’application permet l’échange de messages privés et de groupe, la gestion d’équipes sportives, les notifications et le contrôle des rôles utilisateurs.

Technologies : ASP.NET Core MVC, SignalR, Entity Framework Core, SQL Server, ASP.NET Identity, HTML, CSS, JavaScript, Azure

Tictactoe avec Bot Intelligent

Application Android développée en Kotlin, incluant un bot intelligent et la gestion des états du jeu.

Technologies : Kotlin, Android Studio, Architecture MVVM, Design Pattern Stratégie, Tests unitaires, UI Android (XML)
